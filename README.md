# Annual Report Splitter

## Overview
The Annual Report Splitter is a modernized tool designed to streamline the process of splitting annual reports into manageable sections. This repository contains the source code, usage instructions, and documentation for the tool.

## Features
- **User-Friendly Interface**: The tool offers an intuitive UI for ease of use.
- **Customizable Split Options**: Users can choose how to split reports – by page, section, or custom criteria.
- **Efficient Processing**: The tool utilizes efficient algorithms to handle large report files without significant delays.
- **Output Formats**: Supports multiple output formats including PDF, DOCX, and TXT.

## Installation
To install the Annual Report Splitter, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/ttoomse-prog/annual-report-splitter.git
   ```
2. Navigate to the directory:
   ```bash
   cd annual-report-splitter
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
To use the Annual Report Splitter tool, run the following command:
```bash
node splitter.js --input path/to/input/report.pdf --output path/to/output/ --format pdf
```
- Replace `path/to/input/report.pdf` with the path of your annual report.
- Specify the desired output directory where split files will be saved.
- Choose the format you want for the output files.
